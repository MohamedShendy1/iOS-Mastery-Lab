# Memory Management Deep Dive

## Objective
Understand how ARC works internally and how retain cycles happen in real-world apps.

---

## Concepts Covered

- Automatic Reference Counting (ARC)
- Strong Reference Cycles
- Weak vs Unowned
- Closure Capture Lists

---

## Practical Lab

This folder contains real code examples demonstrating:
- How retain cycles happen
- How to fix them using weak/unowned
- How capture lists affect memory behavior

---

## Production Insight

Memory management decisions directly impact app stability and performance.
